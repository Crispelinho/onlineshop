## Opción Instalacion 1
* Descargar el archivo .zip
* Ejecutar desde la carpeta bin
```
./jmeter
```

## Opción Instalacion 2
install Homebrew
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
update Homebrew
```
brew update
```
install JMeter
```
brew install jmeter
```
